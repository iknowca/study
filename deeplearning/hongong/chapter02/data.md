# data handling

## supervised learning and unsupervised learning

Machine learning algorithms can be broadly categorized into supervised learning and unsupervised learning. Supervised learning requires training data and answers.

In supervised learning, data and answers are called input and target, respectively, and these two are called training data.

In supervised learning, the algorithm learns to predict the answer because there is a target. In contrast, unsupervised learning uses only input data without a target. Since unsupervised learning does not use the answer, it cannot predict anything.

Just as practice questions and test questions must be different to properly evaluate a student's ability, in order to properly evaluate the performance of a machine learning algorithm, the training data and the data used for evaluation must be different.

The data used for evaluation is called the test set, and the data used for training is called the training set.

## sampling bias

If the samples in the training set and test set are not evenly mixed, it means that the sampling is biased in one direction, and this is called sampling bias.
