# closure

클로저란 어떤 함수의 내부 함수가 외부함수의 변수를 참조할 때, 외부 함수가 종료된 후에도 내부 함수가 외부함수의 변수를 참조할 수 있도록 어딘가에 저장하는 함수를 의미한다.

클로저의 조건은 아래와 같다.
* 임의의 함수의 내부함수
* 내부 함수가 외부 함수의 변수를 참조
* 외부 함수가 내부 함수를 리턴
```python
def foo(s):
    v = "Hello " + s
    def bar():
        print(v)
    return bar

temp_f = foo("World")
temp_f()
# Hello World
```

위 코드에서 foo() 함수는 bar() 함수를 리턴한다. 이때 bar() 함수는 foo() 함수의 지역변수 v를 참조한다. foo() 함수가 종료되더라도 bar() 함수는 v를 참조할 수 있다. 이때 bar()함수를 클로저라고 한다.

## 클로저의 장점

전역 변수는 코드 스케일이 커질수록 관리하기 힘들어지는 위험을 내포하고 있다. 하지만, 전역변수는 장점또한 존재하는데, 클로저는 전역변수의 단점을 보완하면서 전역변수의 장점을 활용할 수 있는 방법이다.
