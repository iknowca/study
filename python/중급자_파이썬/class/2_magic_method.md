# Magic Method

special method라고도 불리며, python 내부적으로 builtin 되어 있는 메소드이다.

## 종류

### \_\_new\_\_
새로운 인스턴스를 만들때 호출되며, \_\_init\_\_보다 먼저 호출된다.

### \_\_add\_\_
더하기 연산을 오버로딩 할때 사용된다.

비슷하게 \_\_sub\_\_, \_\_mul\_\_, \_\_truediv\_\_, \_\_floordiv\_\_, \_\_mod\_\_, \_\_pow\_\_ 등이 있다.

### \_\_del\_\_

객체가 소멸될때 호출된다.

### __str\_\_
문자열로 변환할때 호출된다.

### \_\_repr\_\_
문자열로 변환할때 호출된다.
__str\_\_과 \_\_repr\_\_의 차이는 str은 사용자에게 보여줄 문자열을, repr은 개발자에게 보여줄 문자열을 뜻한다.

### \_\_len\_\_
길이를 구할때 호출된다.

### \_\_doc__
문서화 문자열을 뜻한다. docstring을 출력한다.

### \_\_dict\_\_
객체의 속성을 딕셔너리 형태로 출력한다.

### \_\_call\_\_
객체를 함수처럼 호출할때 사용된다.

### \_\_getattribute\_\_
객체의 속성을 가져올때 호출된다.
객체를 바인딩하고 있을때, 점(.)을 통해 접근하면 호출된다.
