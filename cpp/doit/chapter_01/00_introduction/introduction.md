# introduction

## What is C++?
C++ is simply an extension of the C programming language. and called C with classes.

## C++ history

C++ began with a 1979 study called "C with Classes" by Bjarne Stroustrup of AT&T Bell Labs. While C with Classes falls short as a programming language, it completes the specification of C++.

* Class
* Member function and variable access control
* Friend class
* Inline function
* Inheritance
* Constructor and Destructor
* Type checking and conversion
* Operator overloading and default argument

The biggest feature of the above specification is class, which is the core of **OOP**.

The Stroustrup published the first edition of "The C++ Programming Language" in 1985.
And he named the language "C++".

## C++ standard

In 1989, the C++ standard was reorganized with the development of Cfront 2.0.
After the introduction of Cfront 2.0, users began to use C++ more widely and many compilers were developed.

But the problem is that each compiler has its own unique features and bugs. In 1998, the C++ standard was established by the ISO/IEC called "ISO/IEC 14882:1998". This standard is called C++98.

C++98 is the first standard of C++ and is the basis of the current C++ standard. and Template, STL was added in C++98.
## build process

Compiling is the process of converting the source code into code that the computer can understand.
The computer does not understand the source code, so it must be converted into a form that the computer can understand.
This is called object code.

If the source code is many, we need to compile it into multiple object codes and then **link** them together.

C and C++ code need pre-processing, which is a process that is done before the compilation.
The preprocessing process instructions like `#include`, `#define`.
