# transform vs replacement transform

## Transform

The `Transform` class is used to animate the transformation of one mobject into another. The `Transform` class is a subclass of the `ReplacementTransform` class, which is used to animate the replacement of one mobject with another. The `Transform` class is used when you want to animate the transformation of one mobject into another, while the `ReplacementTransform` class is used when you want to animate the replacement of one mobject with another
