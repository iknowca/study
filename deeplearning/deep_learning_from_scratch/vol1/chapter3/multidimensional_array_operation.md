# Multidimensional Array Operation

In deeplearning, multidimensional array computations are commonly used.

## Multidimensional Array

A multidimensional array is an array that has more than one dimension.
```python
import numpy as np

# 1D array
A = np.array([1, 2, 3, 4])
# 2D array
B = np.array([[1, 2], [3, 4], [5, 6]])
```
